# 🛒 Loja Java Spring Boot

Este projeto consiste em uma API backend para gerenciamento de uma loja, desenvolvida com Java e Spring Boot.

## 🚀 Objetivo
O objetivo do projeto é simular o funcionamento básico de um sistema de loja (e-commerce), permitindo o gerenciamento de dados como produtos e possíveis operações relacionadas.

## 🛠️ Tecnologias utilizadas
- Java
- Spring Boot
- Spring Data JPA
- Maven
- Banco de dados relacional

## ⚙️ Funcionalidades
- Cadastro de produtos
- Listagem de produtos
- Estrutura em camadas (Controller, Service, Repository)
- API REST para comunicação com o frontend

## 📂 Estrutura do projeto
O projeto segue o padrão de arquitetura em camadas:
- `controller`: responsável pelas requisições HTTP
- `service`: lógica de negócio
- `repository`: acesso ao banco de dados
- `model`: entidades do sistema
